シナリオ: 総合動作確認テスト

このシナリオでは、Minecraftサーバーの複数の機能を包括的にテストします。

== 接続フェーズ ==
1. サーバーに接続する
2. 接続が成功していることを確認する
3. 1秒待機

== コマンドテスト ==
4. /help コマンドを実行する
5. ヘルプメッセージを受信することを確認する

== 移動テスト ==
6. 座標 (0, 100, 0) にテレポートする
7. 2秒待機
8. プレイヤーが指定座標の近くにいることを確認する

== 状態確認 ==
9. ゲームモードがサバイバルであることを確認する
10. 体力が0より大きいことを確認する

== 終了 ==
11. チャットで「テスト完了」と送信する
12. 1秒待機
13. サーバーから切断する
